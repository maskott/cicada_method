My Bianary Blowout!

Alright, so check it out. This test was inspired by a recent SitePoint post from Alex Walker in which he discussed a new idea for creating seemingly random tiled backgrounds.

Basically, he discussed using multiple varied transparent tile sizes and placements that correspont to prime numbers as layered background images that acheive a far greater "randomness" in a much smaller file size that would be achieveable with a simple pattern graphic.

Let’s break it down…

Take a quick look at the background of this page. All of the green ones and zeros have been modified by extra layers. Only the white digits are showing through from the original tile.

Here is a div filled with only the original tile.

 

And now the fun begins! Here is another div filled with only the first layer and a color to help you see where the transparency in the image is.

 

And now, we add them together…

 

All together, the background of this page is only compsed of 3 fairly small graphic files, but it creates a pattern that doesn't repeat exactly for an unimagineably large grid size. Not a bad idea at all!
